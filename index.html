Medical Coding Automation - CareCloud (Internal)

A multi-agent LLM system automating medical coding inside the Revenue Cycle Management pipeline. Each module (CPT, ICD, Modifier, NCCI Edits, LCD/NCD, MIPS) is handled by a dedicated agent with its own prompt strategy and retrieval flow.

  • Architecture: LangChain + LangGraph multi-agent graph with agent-specific prompts.
  • Retrieval: Semantic search over CPT & ICD codes + descriptions via Vector DB (Pinecone / Qdrant) and PostgreSQL for structured lookup.
  • Knowledge: LCD/NCD article matching, NCCI edits, and MIPS measures as retrievable sources.
  • Fine-tuning: LLaMA 3.3 fine-tuned with LoRA for extraction & diagnosis prediction.
  • Accuracy: CPT 88%, Modifier 86%, MIPS 100%, ICD 70% · actively optimizing ICD.
  • Impact: Reduced manual data processing time by 40%.

ZenBuild · AI-Powered Architectural Design

Final Year Project · Deep Learning & Computer Vision · IST, 2025

An end-to-end AI platform for automated 2D floorplan generation. Built on an enhanced DeepLayout framework (Wu et al., 2019) trained on the RPLAN dataset, with a four-module sequential neural pipeline (Living → Continue → Wall → Location) and a vectorization engine that turns pixel outputs into professional vector floorplans. A Flask web app on top lets users design plans and renders rooms with realistic textures.

  • Up to 99.5% validation accuracy across the four modules.
  • ~8 seconds per design vs ~45 min of manual MATLAB drafting (300× faster).
  • 8/10 architects in user study would use the output as a starting point.
  • Scope: single-story residential, 60–120 m². Supports 2/3/4-room layouts (2_1, 2_2, 3_1, 3_2, 4_1, 4_2).

Bone Fracture Detection

Computer Vision · Two-stage ResNet-50 X-ray classifier

A two-stage CNN pipeline that decides whether an upper-limb X-ray shows a fracture. Stage 1 · a ResNet-50 body-part router picks Hand / Shoulder / Elbow. Stage 2 · a part-specialist ResNet-50 (one per part) outputs fractured / normal. Ships as a CustomTkinter desktop GUI plus an optional Flask web demo.

  • Trained on the MURA dataset (Stanford ML Group) · 20,335 musculoskeletal radiographs.
  • Data augmentation (horizontal flips) + 72/18/10 train/val/test split.
  • Adam optimizer (lr 1e-4) with early stopping.
  • Body-part classification accuracy: 100% · Fracture detection accuracy: 87.9%.

Fine-Grained Bird Classification

Computer Vision · CUB-200-2011 · 200 species · ResNet-50 vs EfficientNet-B4

Benchmarked four configurations on the CUB-200-2011 dataset (stratified 70/15/15 split, fixed seed): ResNet-50 baseline, + bounding-box crop, + MixUp/CutMix, and finally an EfficientNet-B4 trained in two phases (warm-up with frozen backbone → full fine-tune at lower LR with MixUp + CutMix + label smoothing). Grad-CAM confirms the model attends to bird body, head and wing regions · not the background.

  • ResNet-50 baseline → 82.18% Val Top-1 / 95.93% Top-5.
  • + bbox + MixUp / CutMix → 96.55% Test Top-5 (best ResNet variant).
  • EfficientNet-B4 → strongest backbone, best overall.
  • Reproducible pipeline: dedicated scripts for data, training, eval, Grad-CAM and checkpointing.
